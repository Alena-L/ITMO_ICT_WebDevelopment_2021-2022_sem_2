Введение

Цель работы:

Изучение технологий веб-программирования, создание веб-сайта с применением технологического стека Django + Vue JS + Docker

Задачи:

  1. Разработка бэкенда с помощью Django и DRF
  2. Создание интерфейсов для взаимодействия с бэкендом средствами Vue JS
  3. Контейнеризация с использованием Docker

Описание сервиса:

Программная система, предназначенная для администрации аэропорта некоторой компании-авиаперевозчика.
