Add a new airplane

URL : airplanes/create/

Method : POST

Code : 200 OK

Content : {[]}

{
    "tail_number": "",
    "type": "",
    "seats": null,
    "velocity": null,
    "airline": "",
    "under_maintenance": false
}

Method : GET, PUT, PATCH, DELETE
